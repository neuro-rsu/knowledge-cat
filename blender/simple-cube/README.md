# Knowledge Cat — Blender

Проект посвящён созданию 3D-модели Учёного кота для Рязанского государственного университета имени С. А. Есенина.

## Идея проекта

> «Давайте сделаем кота, который под дубом в РГУ сидит. Будет круто».

Текущая сцена с кубом — временная техническая заглушка. В дальнейшем вместо неё должна появиться полноценная 3D-композиция с Учёным котом под дубом на территории РГУ.

## Содержимое

- `simple-cube.blend` — временная сцена-заглушка;
- `preview.png` — изображение текущей сцены;
- `create_scene.py` — скрипт для повторного создания заглушки.

Проект разрабатывается в Blender. Заглушка создана в Blender 5.1.

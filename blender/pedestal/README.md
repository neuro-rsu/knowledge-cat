# Постамент — Blender

Отдельная модель постамента по фотографиям и замерам из каталога `images`. Модель кота и окружающие объекты в сцену не добавлены.

## Состав модели

- нижняя опорная плита;
- переходная плита;
- суженная книзу центральная стойка;
- верхняя опорная плита;
- объёмные буквы «РГУ» на фасаде.

## Основные размеры

- общая высота — `1,02 м`;
- нижняя плита — `0,74 × 0,52 × 0,08 м`;
- стойка — `0,75 м`, от `0,29 × 0,22 м` внизу до `0,35 × 0,26 м` вверху;
- верхняя плита — `0,60 × 0,45 × 0,09 м`.

Размеры, не показанные на фотографиях напрямую, восстановлены по пропорциям конструкции.

## Файлы

- `pedestal.blend` — готовая сцена Blender;
- `preview.png` — превью модели;
- `create_pedestal.py` — скрипт для повторной сборки сцены.

Проект создан в Blender 5.1.
